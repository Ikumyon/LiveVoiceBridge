# LiveVoiceBridge 使い方ガイド

LiveVoiceBridge は、YouTube Live のチャット（コメント）をリアルタイムで取得し、VOICEVOX などの音声合成エンジンを使用して自動で読み上げるデスクトップアプリケーションです。

---

## 1. 事前準備

本アプリを使用する前に、以下の準備を行ってください。

### ① VOICEVOX (または互換エンジン) の起動
1. 読み上げに使用する **VOICEVOX** を PC 上で起動しておきます。
   - ※ VOICEVOX Nemo などを利用する場合は、該当のエンジンを起動してください。
2. アプリの設定画面で、VOICEVOX の接続先 URL（標準では `http://127.0.0.1:50021`）を指定できます。

### ② YouTube Data API v3 キーの取得
コメントの取得には YouTube の API キーが必要です。
1. Google Cloud Console などでプロジェクトを作成し、**YouTube Data API v3** を有効化します。
2. API キー（認証情報）を作成し、コピーしておきます。

---

## 2. 基本的な使い方

### ステップ 1: アプリを起動する
- `LiveVoiceBridge.exe` を実行します（またはスクリプトから起動します）。

### ステップ 2: 初期設定を行う
1. 画面右上にある **設定ボタン（`...`）** をクリックします。
2. **APIキー**: 事前準備で取得した YouTube Data API v3 キーを入力します。
3. **VOICEVOX URL**: VOICEVOX が動作している URL（通常はそのままで問題ありません）を確認します。必要に応じて「VOICEVOX起動テスト」で動作を確認してください。
4. **スピーカー選択**: 「スピーカー選択」ボタンから、読み上げに使うキャラクターと声のスタイル（ノーマル、あまあま等）を選択します。
5. 設定を保存してダイアログを閉じます。

### ステップ 3: 読み上げを開始する
1. **URL / Video ID**: 読み上げを行いたい YouTube Live の配信URL（例: `https://www.youtube.com/live/xxxx`）または動画IDを入力します。
2. **接続開始** ボタンをクリックします。
3. ステータスが「接続中。コメント待機中です。」に変われば、コメントが投稿されるたびに音声で読み上げられます。

### 停止する
- 読み上げを終了する場合は **停止** ボタンをクリックします。

---

## 3. 便利な機能

### ① 別窓（PiP）表示機能
- **「別窓」ボタン** をクリックすると、コメント一覧部分がメインウィンドウから切り離され、単独のウィンドウで表示されます。
- ゲーム配信の画面の上に重ねてコメントをチェックしたい場合に便利です。もう一度ボタンを押すと元の位置に戻ります。

### ② ログ消去機能
- 画面上のコメント履歴およびシステムログをクリアしたい場合は、**ログ消去** ボタンをクリックします。

### ③ チャットコマンド機能
コメント欄に入力する内容によって、音声のパラメータや効果音再生などを一時的にコントロールできます。

#### 1. 辞書登録（教育・忘却）
特定の単語の読み方を学習・削除させます。
- **単語を登録する（教育）**:
  `教育(単語=よみがな)`
  - *例*: `教育(LiveVoiceBridge=ライブボイスブリッジ)`
- **単語を削除する（忘却）**:
  `忘却(単語)`
  - *例*: `忘却(LiveVoiceBridge)`

#### 2. 効果音の再生
`sounds/` ディレクトリに置かれている効果音ファイル（WAV形式など）を再生します。
- **効果音再生**: `再生(ファイル名)` または `音(ファイル名)` または `sound(ファイル名)`
  - *例*: `再生(sound.wav)`

#### 3. 音声パラメータの一時変更
入力したコメントの読み上げ時のみ、声質やエフェクトを変化させます（パラメータ変更後に続く文字列が対象になります）。

- **速度を変更する**: `速度(数値)`
  - *例*: `速度(150)ゆっくりしていってね`（1.5倍速で読み上げ）
- **音程を変更する**: `音程(数値)`
  - *例*: `音程(120)高めの声になります`
- **音量を変更する**: `音量(数値)`
  - *例*: `音量(120)少し大きめの音量です`
- **声の種類（話者）を変更する**: `声(話者ID)`
  - *例*: `声(2)スピーカーを変更します`（※話者IDはVOICEVOXの定義に基づきます）
- **エコーをかける**: `エコー(数値)`
  - *例*: `エコー(50)おーい`（エコーがかかります）
- **やまびこをかける**: `やまびこ(数値)`
  - *例*: `やまびこ(50)ヤッホー`（複数回繰り返すリピートエコーがかかります）
- **定位（ステレオ位置）を指定する**: `左)`, `右)`, `両)`
  - *例*: `左)左から聞こえます` / `右)右から聞こえます` / `両)中央（両耳）に戻ります`

---

## 4. 辞書の管理
設定画面の「読み替え辞書」タブから、登録されている単語一覧の確認・追加・編集・インポート・エクスポート（別グループでの管理）を行うことも可能です。
